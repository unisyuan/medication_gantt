# 🏥 臨床給藥甘特圖系統 (Clinical Medication Gantt Chart System)

本系統專為臨床醫療人員（醫師、藥師、護理師）設計，能將醫院資訊系統（HIS/EHR）複製出的雜亂、未排版原始給藥紀錄數據，在**完全本機端、不需網路的環境下 (100% Offline)**，自動解析並轉換為直觀的互動式給藥甘特圖。

## 🌟 核心特色
1. **完全本機端執行 (100% Offline)**：
   - 無須外網連接。所有解析、計算、繪圖邏輯均於瀏覽器本地執行，符合醫療資安與病患個人隱私（PHD/GDPR）最高安全規範。
2. **📊 互動式藥理分類給藥甘特圖**：
   - 根據藥物的藥理機轉或臨床用途進行自動分組（如：心血管、抗感染、降血糖、中樞神經等）。
   - 一律顯示**學名 (Generic Name)**，方便臨床學術查閱。
   - 繪製精美的 Timeline bar 與代表護理師簽核的給藥時間點圓點。
   - 支援 **「未給藥 (Held/Omitted)」**（如血糖偏低、禁食等）的紅色 `✕` 標記，滑鼠懸停即可顯示詳細原因。
3. **⚙️ 自訂藥物資料庫**：
   - 可在右上角「藥物資料庫」隨時自訂商品名與學名/類別的對應，並儲存於瀏覽器 LocalStorage 中。
4. **📋 實際給藥簽核歷史紀錄表格**：
   - 依時間序列倒序呈現詳細的給藥簽核歷史，包含護理師名稱、劑量、途徑及給藥/未給藥狀態。

## 🚀 如何使用
1. 直接以瀏覽器雙擊開啟 `index.html` 檔案。
2. 在左側面版：
   - 將複製的原始給藥與藥囑資料直接貼在「📥 原始資料輸入」文字框。
3. 右側面版將即時渲染：
   - **藥物給藥甘特圖**與對應的統計指標
   - **實際給藥簽核歷史紀錄表**
